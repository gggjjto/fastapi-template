# FastAPI Best Practices

Review or apply production-grade FastAPI conventions to the current task.

## Workflow

1. Inspect the current code before changing anything.
2. Apply conventions from `.claude/rules/fastapi-best-practices.md` incrementally — avoid rewriting unrelated modules.
3. Prefer consistency with existing patterns when they already match the rules.

## Output

Return one or more of:
- A proposed domain structure or refactor plan with file paths.
- Concrete code changes with a one-line explanation of which convention is applied.
- A review checklist of issues found.

Scaffold the smallest useful structure first; avoid adding infrastructure the project does not need yet.
