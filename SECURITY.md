# Security Policy

## Supported Versions

该仓库以模板形态维护，默认只保证 `main` 分支上的最新代码获得安全修复支持。

## Reporting a Vulnerability

如果你发现安全问题，请不要公开提交 issue 细节。建议通过以下方式私下联系维护者：

- GitHub: [@gggjjto](https://github.com/gggjjto)

请尽量包含：

- 问题描述与影响范围
- 复现步骤（最小可复现）
- 可能的修复建议（可选）

## Response Expectations

- 收到报告后，目标是在 72 小时内确认收到
- 完成初步评估后给出修复计划或缓解方案
- 修复发布后再公开披露细节（如适用）
